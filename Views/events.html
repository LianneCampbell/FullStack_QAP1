Event Emitter

Events The events module allows you to work with events and event-driven programming. It provides the EventEmitter class, which is used to handle asynchronous events.
Usage: Create and handle custom events. Listen for events and execute callback functions when events are emitted. Manage event listeners.

  • EventEmitter - An instance of EventEmitter is created to handle events.
  • eventEmitter.on() - method registers an event handler for a specific event.
  • eventEmitter.emit() - method emits an event, triggering the corresponding event handler.

The use of EventEmitter in this server context provides a clean and decoupled way to handle specific actions when certain events occur.
For example, logging access to pages can be handled by the event listeners, making the main server logic simpler and more focused on routing and serving content.
This event-driven approach can be extended to handle more complex scenarios like authentication events, data processing events, and more.
