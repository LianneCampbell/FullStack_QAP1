Node modules we have used for this project:

HTTP The http module in Node.js allows for the creation of web servers and making HTTP requests. It provides functionalities to handle HTTP requests and responses, making it essential for building web applications and APIs.
Usage: Create an HTTP server that listens for requests on a specified port. Handle incoming HTTP requests and send responses back to the client. Make HTTP requests to other servers.



Filesystem (fs) The fs module provides an API for interacting with the file system. It allows you to read from, write to, and manipulate the file system.
Usage: Read and write files synchronously and asynchronously. Manipulate file paths and directories. Watch for changes in the file system.



Path The path module in Node.js provides utilities for working with file and directory paths. It offers a way to handle and transform file paths in a platform-independent manner, which is particularly useful when working with file systems on different operating systems.
